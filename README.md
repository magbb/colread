# ColRead: Column-based document reader

This is a column-based document reader, inspired by [Tofu](https://amarsagoo.info/tofu/).

ColRead makes it simple to read long documents like in a printed newspaper.

ColRead is made for QT, using PyQt, and written with help from Gemini and Copilot.

## Install (you will need QT5)
pip install .

## Run
python3 main.py

## Use
Simply copy and pase from any program (e.g. a web browser or LibreOffice) into the window.
